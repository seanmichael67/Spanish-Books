🎵 La Música y los Instrumentos 🎸

🎼

¡Vamos a hacer música!

(VAH-mohs ah ah-SEHR MOO-see-kah)

Let's make music!

🎸

Ana toca la guitarra.

(AH-nah TOH-kah lah gee-TAH-rah)

Ana plays the guitar.

Nueva Palabra / New Word:

La guitarra = The guitar

Tocar = To play (an instrument)

🥁

Carlos toca el tambor.

(KAR-lohs TOH-kah el tahm-BOHR)

Carlos plays the drum.

Nueva Palabra / New Word:

El tambor = The drum

🪈

María toca la flauta.

(mah-REE-ah TOH-kah lah FLAU-tah)

María plays the flute.

Nueva Palabra / New Word:

La flauta = The flute

🎹

Luis toca el piano.

(loo-EES TOH-kah el pee-AH-noh)

Luis plays the piano.

Nueva Palabra / New Word:

El piano = The piano

🪇

Sofía toca las maracas.

(soh-FEE-ah TOH-kah lahs mah-RAH-kahs)

Sofía plays the maracas.

Nueva Palabra / New Word:

Las maracas = The maracas

🎶

¡Todos tocan juntos! ¡Es una banda!

(TOH-dohs TOH-kahn HOON-tohs! ehs OO-nah BAHN-dah)

Everyone plays together! It's a band!

Nueva Palabra / New Word:

Juntos = Together

La banda = The band

👂

¿Escuchas la música? ¡Qué bonita!

(ehs-KOO-chahs lah MOO-see-kah? keh boh-NEE-tah)

Do you hear the music? How beautiful!

Nueva Palabra / New Word:

Escuchar = To listen/hear

Bonita = Beautiful/pretty

🎤

¡Ahora vamos a cantar una canción!

(ah-OH-rah VAH-mohs ah kahn-TAHR OO-nah kahn-see-OHN)

Now let's sing a song!

Nueva Palabra / New Word:

Cantar = To sing

La canción = The song

¡Excelente trabajo! 🎉

¡Has aprendido sobre la música!

You've learned about music!
